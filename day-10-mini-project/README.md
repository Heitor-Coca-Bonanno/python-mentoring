# Day 10 — Mini-Project & Showcase

## Structure (2 hours)
- **60 min build**: choose a project, build the smallest working version
- **60 min showcase**: present what you built and explain decisions

## Pick one project
### Option A — CLI Task Manager (SQLite)
Add/complete/delete tasks, list tasks sorted by due date.

### Option B — Trivia Quiz App (SQLite + classes)
Questions in SQLite, category selection, high scores persisted.

### Option C — Budget Tracker (SQLite)
Log income/expenses, summarize by category, show balance.

## Questions you should answer in the presentation
- What is the core data model (tables/fields)?
- What are the main functions/classes and what do they do?
- What was the hardest bug and how did you debug it?
- What feature would you add next?
