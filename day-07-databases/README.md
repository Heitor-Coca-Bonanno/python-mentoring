# Day 07 — Databases (Files + SQLite)

## Why data disappears
Variables live in RAM (temporary). When the program ends, RAM is cleared.

To keep data, write to disk:
- files (JSON/text)
- databases (SQLite)

## SQLite
SQLite is a database stored in a single file.
You can:
- create tables
- insert rows
- query rows
- update/delete rows

## CRUD
Create, Read, Update, Delete — the 4 basic operations every app needs.

## Challenge
Build a contact book backed by SQLite (add/list/search/update/delete).
