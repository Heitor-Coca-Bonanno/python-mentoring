# Day 05 — Collections (Lists & Dictionaries)

## Lists = numbered shelf
- Ordered
- Indexed from 0
- Good for “many similar things”

## Dictionaries = real dictionary
- Look up by key, not position
- Good for “properties of one thing”

## Pattern: list of dictionaries
One dict per item (movie/contact), list holds all items.

## Challenge
Build a movie ranker: add movies, list sorted by rating, find by title, average rating.
